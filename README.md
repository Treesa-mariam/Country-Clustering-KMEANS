# 🌍 Country Clustering using K-Means

 📌 Project Overview
This project clusters countries based on economic and social indicators such as GDP, life expectancy, unemployment rate, CO2 emissions, and access to electricity.

🎯 Objective
To group countries into clusters and analyze development patterns for better policy insights.

## 📊 Dataset
World Bank Economic Indicators Dataset (Kaggle)

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib & Seaborn
- Scikit-learn

## 🔍 Steps Performed
1. Data Cleaning & Missing Value Handling
2. Exploratory Data Analysis (EDA)
3. Feature Scaling
4. Elbow Method for optimal clusters
5. K-Means Clustering
6. Cluster Visualization
7. Insights & Policy Recommendations

## 📈 Key Insights
- Identified clusters representing developed, developing, and underdeveloped countries
- Observed variation between GDP and life expectancy across clusters
- Highlighted infrastructure gaps (electricity access) in certain clusters

## 💡 Conclusion
Clustering helps governments and organizations design targeted economic and social policies.
